# Odoo Real Estate Module

A custom Odoo Real Estate module built during the Odoo Technical Summer Bootcamp.

## Overview

This project demonstrates the basics of Odoo module development using Python models, XML views, access rights, constraints, computed fields, tags, offers, and simple business workflows.

## Features

- Property listing and form views
- Property types and property tags
- Colored property tags
- Computed total area
- Computed best offer price
- Buyer and salesperson fields
- Offers tab with inline offer creation
- Accept and refuse offer actions
- Property status workflow: New, Offer Accepted, Sold, Cancelled
- Sold and Cancelled ribbons
- SQL and Python validation constraints
- Access rights configuration

## Tech Stack

- Odoo 18
- Python
- XML
- PostgreSQL
- Odoo ORM

## What I Learned

Through this project, I learned how Odoo modules are structured and how backend models, frontend views, security rules, and business logic work together in an ERP application.
